## What it does
Stores content that you don't have time to read.
Helps you remember and plan your daily tasks.
Provides suggestions to achieve your daily goals.
Semantic search of stored content.

## How we built it
 * Monolithic architecture.
 * Methodology: Test Driven Development (TDD).
 * Backend: Python + FastAPI.
 * Frontend: React + Vite + Vanilla CSS.
 * LLMs: Ollama + Llama 3 (Open weights).
 * Database: PostgreSQL.
