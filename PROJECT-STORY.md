## What it does
Almacena contenido de distintos formatos que no tienes tiempo de leer. 
Te ayuda a recordar y planificar tus tareas diarias.
Te da sugerencias para lograr tus metas diarias.
Busqueda semántica de contenido almacenado.

## How we built it
Arquitectura monolítica.
Metodología: Test Driven Development (TDD).
Backend: Python + FastAPI.
Frontend: React + Vite + Vanilla CSS.
LLMs: Ollama + Llama 3 (Pesos abiertos).
Database: PostgreSQL.
